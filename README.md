# Guitar Ear Trainer (Godot) üé∏

A powerful, fretboard-centric ear training application built with **Godot Engine 4.x**. This tool is designed specifically for guitarists to bridge the gap between abstract music theory and its physical implementation on the fretboard.

---

## üöÄ Key Features

### 1. Interval Training (with Shape Focus)
- Learn to recognize intervals not just by sound, but by their visual **shapes** on the guitar.
- **Fixed Pos Mode**: Keep the root note consistent to master 1-to-1 relationships between patterns and intervals (e.g., Major 3rd as a 'small diagonal').
- **Cadence Context**: Plays a full I-IV-V-I cadence before each quiz to firmly establish the key in your ear.

### 2. Chord Progression Quiz
- Master functional harmony by identifying common diatonic progressions (I, IV, V, vi, etc.).
- Innovative **Slot-based UI** to track multi-step progressions.
- Answer directly on the fretboard to reinforce position-based thinking.

### 3. Chord Quality & Location
- Distinguish between Major, Minor, Diminished, and Dominant chords.
- Practice finding specific chord voicings across the neck.
- Supports **Guitar Form (CAGED/Drop)** and **Theory** voicing modes.

### 4. Melody Sequencer
- Integrated sequencer to record, edit, and play back melodies.
- Visual note feedback and synchronized audio.

## üõ† Technology Stack
- **Engine**: Godot 4.x
- **Language**: GDScript
- **Audio**: Custom AudioEngine with dynamic MIDI-to-Sample playback.
- **UI**: Modern, responsive Godot UI with a focus on dark-mode aesthetics and "Juice" (visual feedback).

## üì• Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/fruitshoo/eartrainer.git
    ```
2.  **Open in Godot**:
    - Launch Godot 4.x.
    - Click **Import** and select the `project.godot` file in the cloned directory.
3.  **Run**:
    - Press `F5` to start the application.
