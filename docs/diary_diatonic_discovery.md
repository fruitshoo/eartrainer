# 🎵 다이아토닉 코드 퀴즈에서 느낀 것

> 2026년 2월 17일

## 경험

코드 퀴즈를 풀다가 신기한 경험을 했다.

문제가 무작위로 나오는데, 마치 **음악적인 규칙성**을 가지고 출제되는 것처럼 느껴졌다.
단순히 귀에 편한 정도가 아니라, **"다음엔 이 코드가 나오겠지"** 라고 예측한 것이
반복적으로 실현되었다.

너무 놀라서 코드를 뜯어봤다.

## 원인: Diatonic-First Selection

코드 퀴즈는 완전 랜덤이 아니라, **현재 Key와 Mode의 다이아토닉 코드** 중에서 출제한다.
예를 들어 C Major에서는 C, Dm, Em, F, G, Am, Bdim의 7개 코드 풀에서 뽑는다.

그게 전부다. 특별한 AI도, 진행 알고리즘도 없다.
**"올바른 풀에서 뽑는다"는 단 하나의 제약**이 이 모든 차이를 만들었다.

## 왜 예측이 가능했는가?

다이아토닉 코드들은 단순히 "어울리는" 것이 아니라, **기능화성(Functional Harmony)**에 의해 서로 역할을 갖는다:

- **Tonic** (C, Am, Em): 안정감. "도착했다."
- **Subdominant** (F, Dm): 움직임. "어딘가 가고 싶다."
- **Dominant** (G, Bdim): 긴장. "해결해야 한다."

G(긴장) 다음에 C(안정)가 나올 거라고 예측한 건,
내 귀가 수만 곡의 음악에서 무의식적으로 학습한 **화성 패턴**을 기억하고 있었기 때문이다.

7개 중 하나를 고르면 약 1/3 확률로 Tonic이 나오고,
G → C는 1/7 ≈ 14%. 생각보다 자주 일어난다.

## 깨달은 점

1. **다이아토닉 시스템은 간단하지만 강력하다.** 서양 음악의 95% 이상이 이 7개 코드의 조합이다.
2. **내 귀는 이미 화성을 알고 있었다.** 이론을 공부하지 않아도, 평생 들어온 음악이 무의식적 데이터베이스가 되어 있다.
3. **이어트레이닝의 목표**는 바로 이것이다 — *들리는 소리만으로 다음 코드를 예측할 수 있는 귀.*

그리고 오늘, 그게 되고 있었다. 🎯
